# Висновки з тестування REST API на JSONPlaceholder

## Загальна інформація:
- Кількість ітерацій: 3
- Тривалість тесту: 10 секунд
- Кількість запитів: 5 типів (GET, POST, PUT, PATCH, DELETE)
- Загальна кількість виконань: 18

## Результати (Summary Report):
- Усі запити виконані успішно (Response Code: 200 / 201)
- Час відповіді в межах 300–450 мс
- Жодного запиту з помилкою (Error Count = 0)
- Throughput: ~20-25 req/sec

## View Results in Tree:
- GET повернув JSON із userId та title
- POST, PUT, PATCH повертали правильні статуси та body
- DELETE повернув порожнє тіло з 200 OK

## Висновок:
Сервіс JSONPlaceholder успішно обробляє всі типи REST-запитів. Підходить для тренувального та демонстраційного тестування.
